# Changelog

## 1.1.0

- Revamped repository documentation around Akta as the company news API for AI workflows and enterprise news intelligence.
- Added public documentation pages for introduction, getting started, credits, and Company News Monitoring.
- Expanded the Company News OpenAPI schema with article text, company mentions, industry tags, pagination metadata, and the documented response example.
- Added Python and JavaScript examples alongside the cURL example.

## 1.0.0

- Initial OpenAPI 3.1 specification for the Akta company news API.
- Documented the Akta production base URL and API key authentication model.
- Scoped the public spec and examples to the current company news endpoint.
- Added validation workflow and bundled OpenAPI JSON output.
